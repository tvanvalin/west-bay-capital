## SEO Implementation Plan for Rebuild

1. **Unique meta tags per page** — title + description optimized for target keywords
2. **JSON-LD schema** — Organization, LocalBusiness, FinancialService, FAQPage, BreadcrumbList
3. **Dynamic sitemap** — auto-generated from Next.js pages
4. **Proper robots.txt** — allow all, reference sitemap
5. **Blog/content infrastructure** — MDX-based, targeting long-tail keywords
6. **Service pages** — individual pages for each loan type and property type for ranking potential
7. **Local SEO** — full NAP consistency, Google Business Profile recommendation, service area pages
8. **Image optimization** — WebP/AVIF via Next.js Image, descriptive alt text on every image
9. **Internal linking** — contextual links between related content sections
10. **Semantic HTML** — proper heading hierarchy, landmark regions, structured content
